Entendendo os Modelos RGB e CMYK

O que é o modelo RGB?

O modelo RGB (Red, Green, Blue) é um sistema de cores aditivas utilizado principalmente para exibição em dispositivos eletrônicos como monitores de computador, televisores e câmeras digitais. Nesse modelo, as cores são criadas pela combinação das três cores primárias de luz: vermelho (Red), verde (Green) e azul (Blue).

Quando essas três cores são combinadas na maior intensidade possível (255 em cada canal), obtemos a cor branca. Já a ausência dessas cores resulta na cor preta.

Funcionamento do Pixel no RGB

Um pixel (elemento de imagem) em um dispositivo digital usando o modelo RGB é composto por três subpixels: um para cada cor primária (vermelho, verde e azul). Cada subpixel pode variar de 0 a 255 em intensidade, representando diferentes níveis de luz de cada cor.

Por exemplo:

  • RGB(255, 0, 0) = Vermelho puro
  • RGB(0, 255, 0) = Verde puro
  • RGB(0, 0, 255) = Azul puro
  • RGB(255, 255, 255) = Branco
  • RGB(0, 0, 0) = Preto

O que é o modelo CMYK?

O modelo CMYK (Cyan, Magenta, Yellow, Key/Black) é um sistema de cores subtrativas utilizado principalmente em impressão. Diferente do modelo RGB, que é aditivo, o CMYK trabalha com a combinação de cores de tinta para criar outras cores.

No modelo CMYK, a cor preta (K) é usada para melhorar a profundidade e o contraste, pois a combinação de ciano (C), magenta (M) e amarelo (Y) não gera um preto perfeito.

As cores no modelo CMYK são formadas pela redução da luz refletida pelas tintas. Quanto mais tinta é aplicada, mais a luz é absorvida e menos luz é refletida.

Exemplo de combinações:

  • CMYK(0%, 100%, 100%, 0%) = Vermelho
  • CMYK(100%, 0%, 100%, 0%) = Azul
  • CMYK(100%, 100%, 0%, 0%) = Verde
  • CMYK(0%, 0%, 0%, 100%) = Preto
